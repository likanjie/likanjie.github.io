# 准备阶段

  1. 确认是否安装过mysql：

    yum list installed | grep mysql

  2. 删除系统自带的mysql及其依赖命令：

    yum -y remove mysql-libs.x86_64

  3. 确认是否删除干净

    find / -name mysql

  4. 如果存在未删除干净再删除

    rm -rf /var/lib/mysql
rm -rf /var/lib/mysql
rm -rf /usr/lib64/mysql
rm -fr /usr/include/mysql

    注意：卸载后/etc/my.cnf不会删除，需要进行手工删除

    rm -rf /etc/my.cnf

  5. 给CentOS (opens new window)添加rpm (opens new window)源，并且选择较新的源命令：

wget dev.mysql.com/get/mysql-community-release-el6-5.noarch.rpm
yum localinstall mysql-community-release-el6-5.noarch.rpm  
yum repolist all | grep mysql  
yum-config-manager --disable mysql55-community  
yum-config-manager --enable mysql56-community  
yum-config-manager --disable mysql57-community 
yum repolist enabled | grep mysql

如果提示yum-config-manager 无法执行，则安装yum指令工具包：

yum -y install yum-utils

# 安装阶段

  1. 安装mysql 服务器 (opens new window)命令

    yum install mysql-community-server

  2. 启动mysql

    service mysqld start

  3. 查看mysql是否自启动,并且设置开启自启动命令

    chkconfig --list | grep mysqld 2、3、4都是on代表开机自动启动
chkconfig mysqld on 设置开机自动启动

  4. 5.6版本要初始化安全设置：

    mysql_secure_installation

  5. 登录root帐号

    mysql -uroot -p

    输入之前设置的密码

  6. 建立可供远程连接的root用户

    GRANT ALL PRIVILEGES ON *.* TO root@'%' IDENTIFIED BY '你设置的密码' WITH GRANT OPTION;

  7. 刷新权限

flush privileges;
